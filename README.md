# Literalar - Sistema de Busca de Livros (MVP)

Projeto desenvolvido com foco em desenvolvimento ágil, aplicando os conceitos de TDD (Test-Driven Development), BDD (Behavior-Driven Development) e Desenvolvimento Top-Down


### Comportamentos Esperados (BDD)

**Cenário 1: Filtrar por Autor**
* **DADO QUE** o usuário quer ler J.K. Rowling
* **QUANDO** ele envia o filtro de autor
* **ENTÃO** o sistema retorna "Harry Potter e a Pedra Filosofal"

**Cenário 2: Filtrar por Ano**
* **DADO QUE** o usuário busca livros de 1960
* **QUANDO** ele envia o filtro de ano
* **ENTÃO** o sistema retorna "O Sol é para Todos"
